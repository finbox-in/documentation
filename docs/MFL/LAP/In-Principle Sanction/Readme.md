<h1>In-principle Sanctioned Stage</h1>

<p>Case reaches <strong>"In-principle sanctioned"</strong> stage when Credit sanctions the case. Parallel verifications like legal, technical, and RCU might or might not be completed by this time. This stage includes primarily the following activities:</p>

<ul>
  <li>Completion of legal, technical, and RCU verifications is <strong>mandatory</strong> in this stage.</li>
  <li>Actual collateral details updation on <strong>Collateral tab</strong> by BCM after the technical verification is complete.</li>
  <li>Offer negotiation by <strong>BBM</strong>.</li>
  <li><strong>Risk profiling</strong> (risk calculation) is <strong>mandatory</strong> by BCM in this stage, although BCM can also complete this in the <em>Under review</em> stage.</li>
</ul>
